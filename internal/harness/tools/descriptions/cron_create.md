Create a RECURRING scheduled cron job. Cron jobs run repeatedly on a fixed schedule (e.g. every 5 minutes, every hour, daily at midnight). They are NOT one-shot timers — do NOT use cron_create if the user wants to run something once, after a delay, or at a single future time. For one-shot delayed execution, use bash (e.g. 'sleep 120 && command') instead. Write a script using write/bash tools first, then schedule it with cron.
